Customized External Comment Demo

This is a demo of the e-comments self-hosted comment server.

The code was initially developed to provide comments for a 100% static site (http://tweaked.io : server optimization.)

Comments will appear below this area, once loaded via a cross-domain JSONP request. A simple demo shows the default look/feel.

This Demo

This demo is customized in three ways:

  • The URL we request from the comment-server has "/reverse" appended to it - so that the comments come in most-recent-first order.
  • The reply-form is above the comments.
  • The reply-form comes from this page, rather than from the included Javascript.
